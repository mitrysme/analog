MDIWindowManager Samples ReadMe

1) Start by inspecting MainFormSimple. It shows the most basic 
usage of MDIWindowManager. 

2) MainFormAdvanced shows some more advanced techniques such 
as using the Auxiliary Window, cloning the MDIWindowManager
"Windows" menu, displaying various orientations, and
supporting Classic MDI easily in the same application.

3) MainFormAltTabs demonstrates using a Custom TabProviders to
replace the MDIWindowManager's built-in tabs.

4) MainFormCustomPaint shows how you can handle the TabPaint
event to draw tabs.
